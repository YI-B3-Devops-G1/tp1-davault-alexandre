# Mode opératoire


## Générer la VM

Depuis un terminal (dans le répertoire du dossier) :
	> vagrant up

Puis :
	> vagrant ssh


## Supprimer la VM

Depuis un terminal (dans le répertoire du dossier) :
	> vagrant destroy
